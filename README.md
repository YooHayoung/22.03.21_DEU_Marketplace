# DEU Marketplace
> 동의대학교 학생들을 위한 중고 거래 서비스 플랫폼입니다.
> 잘 사용하지 않는 지난 학기의 전공 서적, 일반 서적, 기타 중고 물품들을 교내에서 거래할 수 있습니다.

## 프로젝트 기간
- 2022.03 ~ 2022.05

## 기술 스택
### BackEnd
- Java 11
- SpringBoot 2.6.6
- JPA
- QueryDsl
- MySQL
- Spring Security

### FrontEnd
- JavaScript
- React.js

### 배포
- AWS EC2
- AWS S3 : 이미지 등 정적 호스팅
